## About this project

This project represents my professional online CV and serves as part of my developer portfolio.

It demonstrates:

- clean semantic HTML5 structure
- modern CSS layout using Flexbox
- responsive design principles
- deployment using Azure Static Web Apps
- CI/CD workflow with GitHub Actions

---

## Technologies used

- HTML5
- CSS3
- Azure Static Web Apps
- GitHub Actions
- Git & GitHub

---

## Purpose

The purpose of this project is to:

- present my professional profile online
- showcase my frontend and web development skills
- demonstrate my ability to deploy applications using cloud services
- build a strong developer portfolio

## Deployment

This website is automatically deployed using Azure Static Web Apps.

Each update pushed to the `master` branch triggers automatic deployment via GitHub Actions.
